Semantica

E sua importancia no HTML 5

O HTML 5 pode significar palavras internamente ao invez de só alteralas estecamente
oque é importante para o funcionamento de alguns sites etc. para isso usa-se algumas tag`s

Algumas tag´s Semanticas

  1. strong
  2. em
  3. mark
  4. small
  5. del
  6. ins

Exitem varias outras

Tag "strong"

Indica força dentro da frase, oque a deixa em negrito Satoro Gojo é o mais forte

Tag "em"

Significa enfase, o termo aparece em destaque Entre o ceu e a terra eu sou o honrado

Tag "mark"

Significa marcação como se fosse um canetão mesmo, use style para mudar a cor Tecnica do vazio: roxo

Tag small

Traz a ideia que o pedaço marcado é "menor" e o afeta visualmente Expansão de Dominio: Santuario Malevolente

Tag Del

Indica que o texto deve ser desconciderado Sukuna é o personagem mais forte de Jujutsu Kaisen

Tag ins

Atribui importancia ao texto, esteticamente o sublinha Aquele que são incrivelmente forte devem permanecer sozinhos

Tag Code

Significa que o texto na tag é um codigo de programação public class OlaMundo { public static void main(String[] args) { System.out.println("Olá, Mundo!"); } }
